风车邮箱 v1.5

使用说明：
1. 双击fengche_mail.exe运行程序
2. 登录或注册账号
3. 在主界面设置您的邮箱地址
4. 点击"开始收信"接收邮件

新功能：
- 收件箱内容自动提取6位数字验证码，支持带空格的验证码
- 优化了对Cursor和Windsurf等邮件的验证码提取
- 邮件内容只显示验证码，不显示多余内容
- 查看邮件时可以点击"复制验证码"按钮直接复制验证码
- 新增"查看原始邮件"按钮，可以查看邮件的原始内容
- 修复了"开始收信"按钮第二次点击不重新拉取邮件的问题

注意事项：
- 每点击开始收信将消耗1点配额
- 配额用完后将无法继续接收邮件
- 点击"充值配额"按钮可以购买更多配额
